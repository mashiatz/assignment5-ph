Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use various methods depending on your needs:

getElementById(): Selects a single element with a specific id. Returns the element object.

getElementsByClassName(): Selects all elements with a specific class name. Returns an HTMLCollection, which is similar to an array but not exactly the same.

getElementsByTagName(): Selects all elements with a specific tag name (like div, p, etc.). Returns an HTMLCollection.

querySelector(): Selects the first element that matches a given CSS selector (e.g., by tag, class, id, or attribute).Returns the first matching element.

querySelectorAll(): Selects all elements that match a given CSS selector. Returns a NodeList, which is an array-like collection that can be looped through.

What is event delegation in the context of the DOM and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

How do you manipulate an element's attributes and styles using the DOM?

The classList object only provide methods to change the class of an element. If you want to change other attributes like id , href , or src , you can use the setAttribute() method. The setAttribute() method accepts two arguments: The name of the attribute to set.
